# ConsumptionPlan v3.1（Resume Edition）

## 1. 项目概述

### 1.1 项目名称

ConsumptionPlan —— AI智能消费分析与预算管理平台

### 1.2 项目背景

随着移动支付的普及，人们的消费行为越来越频繁，但大多数用户缺乏系统化的消费记录与分析工具，难以及时了解自己的消费结构和财务状况。

本项目旨在帮助用户记录日常收支情况，通过数据可视化分析、预算管理以及AI消费报告等功能，帮助用户养成科学消费习惯，提高个人财务管理能力。

### 1.3 项目目标

构建一个集：

- 消费记录
- 数据统计
- 预算管理
- AI分析报告
- 记账打卡
- 通知提醒
- 用户反馈

于一体的个人消费管理平台。

# 6. 项目亮点

## 亮点一

基于 JWT 实现用户认证与权限控制。

## 亮点二

采用 Vue3 + Spring Boot 前后端分离架构。

## 亮点三

使用 ECharts 实现消费数据可视化分析。

## 亮点四

集成 DeepSeek API 自动生成个性化消费分析报告。

## 亮点五

设计预算管理与超支提醒机制。

## 亮点六

设计记账打卡功能，提高用户持续记账率。

## 亮点七

实现通知中心与用户反馈机制，提高系统完整性。
